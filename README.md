## Run Locally

Clone the project

```bash
  git clone https://github.com/rajujai/youtube_clone.git
```

Go to the project directory

```bash
  cd youtube-clone
  code .
```

Preferered IDE :

  VS-Code
  Install Live-Server Extension


Start the server :

  Right click on index.html and 
  open with Live-Server
